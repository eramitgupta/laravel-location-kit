# Contributing

Thanks for contributing to **Laravel Location Kit** 🚀

## Setup

```bash
git clone https://github.com/your-username/laravel-location-kit.git
cd laravel-location-kit
composer install
```

Frontend:

```bash
cd vue && npm install
cd ../react && npm install
```

## Guidelines

* Follow PSR-12
* Keep code clean and simple
* Avoid breaking changes

## Pull Requests

```bash
git checkout -b feature/your-feature
git commit -m "feat: your change."
git push origin feature/your-feature
```

Open a PR with clear details.

## Issues

Report bugs or suggest features via GitHub issues.
